# Split Bills

A simple utility to split monthly bills between flatmates.

## Features
- Track fixed and variable bills (Rent, WiFi, Water, Electric, etc.)
- Calculate total amount and split it 50/50.
- Export a professional PDF summary to share.
- Local storage persistence.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
